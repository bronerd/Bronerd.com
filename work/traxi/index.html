Traxi | Vehicle Tracking Apps

Project info
Client: Shockoe.com Traxi.co
Work Description

As a part of a vehicle tracking application we were developing at Shockoe.com Traxi was born. Packaged as both a web and mobile application Traxi is the perfect solution for any bus or shuttle service looking to connect with their users. Using the latest Google maps API, and GPS hardware installed inside the vehicles, we were able to create a reliable location tracking solution. As project lead for Traxi I helped create a pretty unique user interface for both web and mobile platforms. I also had the pleasure of branding Traxi, with a simple block font and vibrant colors.
